Welcome to PubKey Service!

The service is used to share public keys used for secure encryption and verification of messages.
This is an open-source project, so source code of this service You can found here: https://github.com/marcinbor85/pubkey.
The website works with the NES Messenger application, which is also open-source project: https://github.com/marcinbor85/nes.
This software is released under the MIT license.

API:

  • GET /users/<username>

    Get user public key

    • 200 - OK
      								{
									    "public_key": "<PUBLIC_KEY_PEM_FORMAT>"
								}
    • 404 - User not found

  • GET /users/<username>/<token>

    Security token to activate and deactivate user

    • 200 - OK
    • 404 - User/Token not found

  • POST /users

    Register new user

    						{
							    "username": "<USERNAME>",
							    "email": "<EMAIL>",
							    "public_key": "<PUBLIC_KEY_PEM_FORMAT>"
						}
    • 201 - OK
    • 400 - Bad request
